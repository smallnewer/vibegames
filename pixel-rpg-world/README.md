# Pixel RPG World

一个基于 Babylon.js、React 和 TypeScript 构建的像素风动作 RPG 原型，包含地下城探索、战斗、技能、装备、锻造、本地多人控制和存档系统。

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
npm install
npm run dev
npm run build
```

## Useful Commands

- `npm run dev`: start local development
- `npm run build`: build the production bundle
- `npm test`: run content, icon, unit and production-build checks
- `npm run balance:simulate`: run the deterministic balance simulation
